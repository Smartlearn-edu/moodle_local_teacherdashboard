# Changelog

All notable changes to the **Smart Dashboard** plugin will be documented in this file.

## [1.0.0] - 2026-02-15

### Added
- **Overview section** with course cards, student counts, and pending grading indicators
- **Student Progress section** with cross-course completion tracking and detailed per-student view
- **Grading section** with pending submissions overview across all courses
- **Analytics section** (Admin/Manager) with system-wide enrollment stats, category breakdowns, and ratio charts
- **Payment Analytics section** (Admin/Manager) with revenue tracking, time-range filtering, and per-category breakdowns
- **Settings section** with payment mode and currency display configuration
- **Dark mode UI** with modern sidebar navigation
- **CSV export** for Student Progress, Analytics, and Payment Analytics
- **Hierarchical category filtering** with recursive subcategory support
- **Responsive design** that works across desktop and tablet
- Support for Moodle 4.0 through 4.5
